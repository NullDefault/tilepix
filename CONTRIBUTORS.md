## Style guide
The following lists should be edited in alphabetical order based on **username**.

The format of each line must be:
```
[Name] @[github username] <[email (optional)]>
```

For example:
```
Joe Bloggs @joebloggs joe@bloggs.com
Jane @asdf1234
```
